# Visual C++ Redistributable

В эту папку нужно поместить установщик Visual C++ Redistributable 2015-2022 x64.

## Как получить файл:

1. Скачайте с официального сайта Microsoft:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

2. Или используйте прямую ссылку:
   https://aka.ms/vs/17/release/vc_redist.x64.exe

3. Сохраните файл как `vc_redist.x64.exe` в эту папку

## Альтернативный вариант:

Если файл не включен, установщик попытается скачать его автоматически с официального сайта Microsoft.

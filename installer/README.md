# Установщик 1C Updater Agent

Графический установщик для установки службы 1C Updater Agent.

## Особенности

- ✅ Графический интерфейс для ввода параметров
- ✅ Автоматическая проверка прав администратора
- ✅ Автоматический поиск exe файла агента
- ✅ Детальное логирование всех операций
- ✅ Лог файл сохраняется рядом с установщиком
- ✅ Прогресс установки
- ✅ Обработка ошибок

## Сборка

```bash
cd installer
dotnet build -c Release
```

Результат: `bin/Release/net8.0-windows/1CUpdaterAgentInstaller.exe`

## Использование

1. Скопируйте `1CUpdaterAgentInstaller.exe` и `1CUpdaterAgent.exe` в одну папку
2. Запустите `1CUpdaterAgentInstaller.exe` от имени администратора
3. Введите ID ПК и URL сервера
4. Нажмите "Установить"
5. После установки проверьте лог файл рядом с установщиком

## Лог файл

Лог файл создается автоматически рядом с установщиком с именем:
`1CUpdaterAgentInstaller-Install-YYYYMMDD-HHMMSS.log`

## Требования

- Windows 10/11 или Windows Server
- .NET 8.0 Runtime
- Права администратора
